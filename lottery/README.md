# 網頁版抽籤程式

純前端(HTML / CSS / JavaScript)的隨機抽籤工具,無需安裝、無需後端伺服器。

## 功能特色

- 📝 **自訂名單**:每行一個名字或項目,即時顯示數量
- 🎯 **可設定抽出數量**:一次抽出 1 至多名
- 🔁 **抽過不重複**:可選擇被抽中者自動從名單移除(適合分組、抽獎)
- 🎰 **滾動開獎動畫**:增加抽籤臨場感
- 🕑 **抽籤紀錄**:保留每次開獎結果與時間
- 📱 **響應式設計**:電腦、手機皆可使用

## 使用方式

1. 用瀏覽器直接開啟 `index.html`
2. 在左側輸入名單(每行一個項目),或點「載入範例」
3. 設定抽出數量,按下「🎯 開始抽籤」即可

## 技術說明

- 使用 **Fisher–Yates 洗牌演算法**確保隨機公平
- 單一檔案、零相依套件,適合作為 AI / 程式設計教材範例
